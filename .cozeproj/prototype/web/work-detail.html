Aura 设计系统

为现代 SaaS 产品打造的综合性设计系统，涵盖色彩规范、组件库与交互模式。

React TypeScript Tailwind CSS Figma Storybook Radix UI

Aura 诞生于对一致性与效率的追求。在快速迭代的 SaaS 开发环境中，设计碎片化是团队常面临的难题。Aura 通过建立一套完整的视觉语言和可复用组件体系，帮助设计师与开发者在同一套规则下协作，显著缩短从设计稿到可用界面的转化周期。

整个系统以"克制与清晰"为核心美学原则。色彩方案采用中性灰阶为主轴，辅以低饱和的品牌强调色，确保信息层级在任何背景下都清晰可辨。 typography 层级经过精心调试，标题使用 Playfair Display 衬线字体营造杂志般的阅读质感，正文则使用 Noto Sans SC 保证屏幕可读性。

组件层面，Aura 提供了超过 60 个原子级组件和 20 余个复合模板，覆盖表单、数据展示、导航、反馈等全部核心场景。每个组件均支持深色模式、多尺寸变体和无障碍访问，并配有详细的 Storybook 文档与使用示例。设计令牌（Design Tokens）与代码实现保持同步更新，确保设计意图能够无损传递至最终产品。
